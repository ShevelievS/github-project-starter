# C++ Project

A C++ project created with GitHub Project Starter.

## Requirements

- C++17 compiler (GCC 7+, Clang 5+, MSVC 2017+)
- CMake 3.10 or higher

## Build
```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Run
```bash
./build/app
```
