# Java Project

A Java project created with GitHub Project Starter.

## Requirements

- Java JDK 11 or higher
- Maven 3.6 or higher

## Build
```bash
mvn clean compile
```

## Run
```bash
mvn exec:java
```

## Test
```bash
mvn test
```

## Package
```bash
mvn package
```

## Project Structure
```
.
├── src/
│   └── main/
│       └── java/
│           └── com/
│               └── example/
│                   └── Main.java
├── pom.xml
├── README.md
└── .gitignore
```

## Next Steps

1. Modify package name in `pom.xml`
2. Update `Main.java` with your code
3. Add dependencies in `pom.xml`
4. Create tests in `src/test/java/`
